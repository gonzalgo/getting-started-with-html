The Basics of the Web
There are 5 basic components to understand about the web:

1. The World Wide Web - WWW - HTML (Hypertext Markup Language) or hyperlinks that connect the web; invented in the early 1990's.
Major pieces of the web include: You, your computer/browser, the internet, and servers

HTML - Hypertext Markup Language - made up of:
- text content = what you see
- mark up = what it looks like
- references to other documents (e.g videos and images like this one )
- links to other pages

There are a variety of tags that can be associated with HTML markup - collectively called "elements"
b = bold
em = italicize

There are also a variety of attributes that can be added to HTML markup to further enhance the capability of a webpage
The "a href" tag is used to link other webpages to yours. It can be used to link sites like Reddit or CNN.
Images can also be added using the "img" tag.

The use and understanding of whitespace is important - it allows you to place text on multiple lines.

Two types of elements to understand are INLINE and BLOCK elements:
- Examples of INLINE elements include the use of the following tags: b, em, img, br, and span
- Examples of BLOCK elements include the use of the following tags: p, and div

**INLINE tags behave differently from BLOCK tags; BLOCK elements can add height and width to a web page

Tags
- HTML tag is contained within angled brackets
- most tags have an opening and closing tag
- some tags are void or do not require a closing tag - i.e. the "br" tag

Element
- An HTML element refers to everything within a set of opening and closing tags

Attribute
- This is a property of an HTML element. i.e. - to set the HREF attribute of an anchor tag to the Udacity URL you would write a href="www.udacity.com"
